Тема: Приложение для дистанционного образования

Функционал: 
- Авторизация текущих и регистрация новых пользователей

- Личный кабинет пользователя

- Поддержка различных ролей (читатель, редактор)
   - просмотр расписания и конспектов (доступно читателю)
   - добавление конспектов, пар в расписание (доступно редактору)

- Расписание занятий
   - отображение расписания на неделю
   - добавление нового занятия 
   - связь с материалами предметов по дате (переход на нужные материалы по дате)

- Разделы с различными предметами
   - добавление нового раздела 
   - деление раздела на лекционные занятия и практические
   - загрузка материалов в приложение 

- Конспекты 
   - разделение конспектов по дате/теме
   - связь конспектов по теме с контрольными вопросами на экзамене 

- Онлайн-конспектирование
   - форма для написания конспекта
   - обрезание снимков экрана по заданному формату

- Статистика 
   - процент выполненных домашних заданий
   - процент прослушанного курса от полного 

- Напоминание с отслеживанием дедлайнов
